# Personal Portfolio Website 🌐

This is my personal portfolio website created using HTML, CSS, and JavaScript.  
It showcases my skills, projects, education, and contact details in a modern and interactive design.

## ✨ Features
- Responsive design for mobile and desktop
- Smooth scrolling and animations
- Skills and projects section
- Interactive terminal section
- AI chatbot feature
- Dark and light mode toggle
- Contact section with social links

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Sections Included
- Home
- About
- Skills
- Projects
- Contact

## 🚀 Purpose
This portfolio was built to improve my frontend development skills and to showcase my work, creativity, and learning journey as a developer.
